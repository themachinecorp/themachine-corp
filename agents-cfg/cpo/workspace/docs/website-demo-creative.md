# THEMATHINK 官网产品演示创意方案

## 一、核心概念

**演示主题**：让用户"看见 AI 的思考过程"
**Slogan**：Thinking in Motion
**核心体验**：交互式展示 AI 的深度思考，而非静态展示功能

## 三、视觉设计

### 3.1 视觉风格

| 元素 | 设计 |
|------|------|
| 主色调 | 深空蓝 + 星光白 |
| 强调色 | 琥珀金（思考之光） |
| 动效 | 粒子流动、思维连线 |
| 字体 | 清晰易读 + 等宽字体（代码感） |

### 3.2 动画效果

**思维流动**：
- 输入问题 → 粒子向中心汇聚
- 思考中 → 粒子扩散形成思维网络
- 结果展示 → 光芒渐显，层次展开

**交互动效**：
- 按钮悬停：微光扩散
- 卡片展开：折叠动画
- 思维分支：生长动画

### 4.3 思维导图

```typescript
// 可展开的思维节点
const ThoughtNode = ({ node, depth = 0 }) => {
  const [expanded, setExpanded] = useState(false);
  
  return (
    <motion.div
      animate={{ 
        opacity: expanded ? 1 : 0.7,
        scale: expanded ? 1.05 : 1 
      }}
    >
      <NodeContent>{node.content}</NodeContent>
      
      {node.children && (
        <motion.button
          onClick={() => setExpanded(!expanded)}
          animate={{ rotate: expanded ? 180 : 0 }}
        >
          ▶
        </motion.button>
      )}
      
      {expanded && node.children.map(child => (
        <ThoughtNode node={child} depth={depth + 1} />
      ))}
    </motion.div>
  );
};
```

## 五、演示流程

### 完整演示路径（60秒）

| 时间 | 场景 | 动作 |
|------|------|------|
| 0-5s | Hero | 粒子背景 + Slogan 浮现 |
| 5-10s | 邀请 | "输入一个问题" 提示 |
| 10-20s | 输入 | 用户键入问题（演示预设） |
| 20-35s | 思考 | 思维网络生长动画 |
| 35-50s | 展示 | 多维思维导图展开 |
| 50-60s | 收尾 | CTA "开始体验" |
