## 三、核心工具详解

### 3.1 DeepThink (旗舰工具)

**定位**: 深度思考引导器

**核心逻辑**:
```
用户输入问题 → AI不直接给答案 → 而是追问 → 用户回答 → 继续追问 → 最终用户自己找到答案
```

**交互流程**:
1. 你问："我该不该创业？"
2. AI："你为什么想创业？"（不回答创业好不好）
3. 你答："因为不想打工"
4. AI："打工让你最不舒服的是什么？"
5. ... (5-10轮追问后)
6. 用户自己得出结论

**品牌价值**: 完美体现 "We don't give answers. We help you find your own."

---

### 3.2 Value Finder

**定位**: 价值观发现器

**核心逻辑**:
- 给你两个场景选择
- 每次选择揭示你的一部分价值观
- 10轮选择后，生成你的价值观图谱

**输出示例**:
```
你的核心价值观排序:
1. 自由 > 稳定
2. 成长 > 安全
3. 深度 > 广度
4. 创造 > 执行
```

---

### 3.3 Agent Config

**定位**: Agent配置生成器

**功能**:
- 选择Agent类型（研究者/写手/分析师/客服）
- 填写基础配置
- 生成完整的system prompt
- 支持一键导出到OpenAI/Anthropic格式
