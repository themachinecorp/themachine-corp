## 四、页面结构

1. **Header** - Logo + 导航 + 状态指示
2. **Hero** - 监控网格背景 + 扫描光效 + 大标题
3. **Philosophy** - 三大理念卡片 (观察/引导/觉醒)
4. **Capabilities** - 技术能力展示 + Agent演示
5. **Vision** - "No employees. Just AI agents."
6. **Footer** - 链接 + 版权

---

## 五、动画特效

### 5.1 背景特效

- **监控网格**: 金色细线网格，缓慢呼吸闪烁
- **数据流**: 垂直光线从上到下流动（金色）
- **随机闪烁点**: 黑色背景上随机金色光点

### 5.2 扫描光效

- 页面加载时，**金色**扫描线从顶到底
- 类似安防监控的扫描效果

### 5.3 文字动画

- **打字机效果**: 标题逐字打印显示
- **逐行淡入**: 副标题逐行淡入上移

### 5.4 交互特效

- **摄像头悬停**: 鼠标悬停时旋转
- **卡片发光**: 悬停时金色边框发光
- **按钮光效**: 悬停时光带流动

## 六、组件规范

### 按钮

- 主按钮: 琥珀金背景 + 斜角裁切
- 次按钮: 透明 + 金色边框

### 卡片

- 黑色背景 + 金色边框(角落装饰)
- 悬停时发光效果

### 终端框

- 黑色背景 + 金色边框
- 顶部三个圆点 (红/黄/金)

### 监控指示器

- 小圆点 + 文字
- 圆点金色闪烁

---

## 七、页面加载动画序列

```
0.0s: 黑屏
0.5s: 扫描线扫过
1.0s: 监控网格淡入
1.5s: 标题打字机效果
2.5s: 副标题淡入
3.0s: CTA按钮滑入
3.0s+: 监控点开始闪烁
```
