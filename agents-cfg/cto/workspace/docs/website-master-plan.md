## 二、视觉设计规范

### 色彩系统

| 用途 | 颜色 | Hex |
|------|------|-----|
| 背景主色 | 深渊黑 | #0A0A0F |
| 强调色/标题 | 暗金 | #C9A227 |
| 高亮/交互 | 全视之眼金 | #FFD700 |
| 成功/数据 | 矩阵绿 | #00FF41 |
| 危险/告警 | 警告红 | #FF3131 |
| 卡片/分割 | 钢铁灰 | #1A1A2E |

### 标志性视觉

1. **全视之眼 (The All-Seeing Eye)**
   - 页面中央的 AI 眼睛动画
   - 瞳孔随鼠标移动变化
   - 底部滚动监控数据

2. **扫描线覆盖层**
   - CRT 电视风格的移动扫描条纹
   - 页面加载时的启动序列

3. **浮动代码流**
   - 背景加密风格的滚动数据
   - 像黑客帝国一样的数字雨

4. **威胁等级卡片**
   - 每个痛点的威胁百分比
   - 从绿到红的渐变

### 动效序列

```
0.0s - 黑屏
0.5s - 屏幕闪烁 + 扫描线出现
1.0s - "INITIALIZING..." 打字机效果
1.5s - 眼睛图案落下
2.0s - 金色标题淡入
2.5s - 数据卡片依次弹出
3.0s - 扫描线消失，正式显示
```

## 三、十大骚功能

### 1. 痛点实时直播间 ⚡
- **功能**: 实时抓取 HN/Reddit/GitHub 新痛点
- **体验**: WebSocket 推送，像看直播一样
- **交互**: 弹幕评论 + 打赏"已阅"
- **价值**: 第一时间发现新趋势

### 2. 痛点热度地图 🗺️
- **功能**: 全球痛点地理分布可视化
- **数据**: 按国家/城市筛选
- **交互**: 点击区域看详情
- **价值**: 区域市场机会

### 3. AI 预测水晶球 🔮
- **功能**: 预测未来 7 天热门痛点
- **能力**: 
  - "下周三 GitHub 会火什么？"
  - "下个月 AI 领域最大痛点？"
- **输出**: LLM 生成预测理由
- **价值**: 先人一步

### 4. 痛点 + 解决方案配对 🤝
- **功能**: 痛点自动匹配开源项目
- **能力**: 
  - "这个痛点已经被 X 项目解决"
  - 匹配度评分
  - 替代方案推荐
- **价值**: 快速验证可行性

### 5. 社区验证引擎 🗳️
- **功能**: 用户投票验证痛点真实性
- **算法**: 热度 = (讨论数 × 投票分) / 时间
- **机制**: 防止刷分 + 实名认证
- **价值**: 过滤噪音

### 6. 痛点实验室 🔬
- **功能**: 提交假设痛点，AI 验证
- **流程**:
  1. 用户提交假设
  2. AI 搜索全网验证
  3. 返回结果
- **结果**:
  - 🔴 无人问津
  - 🟡 有讨论
  - 🟢 已解决
- **价值**: 低成本验证想法

### 7. 痛点时间轴 📅
- **功能**: 痛点演变历史
- **案例**: "Prompt injection 痛点的 2024-2026"
- **能力**: 追溯起源 + 追踪演变
- **价值**: 理解趋势周期

### 8. 竞品痛点雷达 🎯
- **功能**: 监控竞品 GitHub/社区
- **数据**: 用户抱怨自动提取
- **输出**: 对比表格 + 趋势图
- **价值**: 竞品监控

### 9. API + Webhook 🔌
- **功能**: 开放痛点数据接口
- **能力**:
  - REST API
  - GraphQL
  - Webhook 推送
- **场景**: 集成到自己产品
- **价值**: 开发者生态

### 10. 痛点 NFT (概念) 🎨
- **功能**: 每个痛点 mint 一次
- **意义**: 记录在链上
- **归属**: "发现者" 身份
- **状态**: 概念验证

## 六、技术架构

### 前端
- **框架**: Next.js / React
- **动画**: Framer Motion
- **地图**: Mapbox / D3.js
- **状态**: Zustand

### 后端
- **API**: Node.js + Fastify
- **实时**: WebSocket
- **数据库**: PostgreSQL + Redis
- **搜索**: Elasticsearch

### AI
- **预测**: LLM (GPT-4 / Claude)
- **验证**: RAG + 全网搜索
