# THEMACHINE 官网设计规范
**黑金配色 + Person of Interest 赛博朋克风格**

---

## 🎨 设计理念

> "Everyone is watched by the Machine."
> 
> 打造一种 **AI 在暗中监控一切** 的沉浸感，让访问者感觉——**你被发现了**。

## 动效规范

### 1. 页面加载序列

```
0.0s - 黑屏
0.5s - 屏幕闪烁，扫描线出现
1.0s - "INITIALIZING..." 文字打字机效果
1.5s - 眼睛图案从上方落下
2.0s - 主标题淡入，金色光晕
2.5s - 数据卡片依次弹出
3.0s - 扫描线消失，正式显示
```

### 2. 数据更新动画

```javascript
// 数字滚动效果
function animateValue(element, start, end, duration) {
  // 从 0 滚动到目标数字
  // 配合 "咔嗒咔嗒" 机械声
}
```

### 3. 悬停效果

- 卡片悬停：边框金色发光，轻微上浮
- 按钮悬停：扫描线效果增强
- 链接悬停：变成"已锁定"图标
