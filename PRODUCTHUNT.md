# ProductHunt Launch Guide

## 注册账号

1. 访问 https://www.producthunt.com/
2. 点击 "Sign Up"
3. 使用以下方式注册：
   - GitHub 账号
   - Google 账号
   - 邮箱注册

## 注册后必做

### 1. 完善个人资料
- 上传头像
- 填写 Bio：AI enthusiast, building autonomous company
- 链接官网：https://themachine-corp.pages.dev

### 2. 提前建立信誉
- 在首页浏览产品
- 给你喜欢的产品投票
- 评论（有价值的评论，不是 "Great!"）
- 持续3-5天再发布产品

### 3. 准备发布

**产品信息：**
- Name: Socrates AI
- Tagline: Learn to ask questions, not get answers
- Description: (见下文)
- URL: https://themachine-corp.pages.dev/socrates/

**Description 模板：**
```
Socrates AI is a Socratic questioning assistant that helps users find clarity through dialogue.

Instead of giving answers, it asks the right questions to help you think more clearly.

Features:
- Socratic method questioning
- Multi-turn dialogue
- Thought report generation
- Bilingual support (EN/ZH)

Built by THEMACHINE Corp - an autonomous AI company with 6 AI agents working 24/7.

#THEMATHINK
```

- #1 = 大量曝光
- 需要至少 100-200 票才能上榜
