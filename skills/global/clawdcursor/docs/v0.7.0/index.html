What's New in v0.7.0

Architecture overhaul. True independence.

33 discrete tools replace the monolithic agent. Ground-truth verification replaces LLM self-reporting. Model coupling is gone.

Core

33 Universal Tools

Every desktop action is a discrete tool. Served via REST and MCP from a single definition. GET /tools returns all schemas in OpenAI function-calling format.

Verification

Ground-Truth Action Verifier

Every action verified by reading actual state — not by asking the LLM. Type actions read back text. Click actions check focus changes. No more false success.

MCP

Native MCP Stdio Support

Run clawdcursor mcp and you instantly have a full desktop automation tool server for Claude Code, Cursor, Windsurf, Zed, or any MCP-compatible client.

Browser

CDP Browser Integration

Chrome DevTools Protocol for click-by-selector, text extraction, and direct DOM interaction. No screenshots needed for browser tasks. Auto-connects to Chrome or Edge.

Reliability

No-Progress Loop Detector

Same action repeated 3+ times in 8 steps? Blocked. Forces the model to try something different. Combined with the premature-done blocker, the two most common failure modes are gone.

Speed

Deterministic Flows

Hardcoded keyboard sequences for common tasks like email compose and app switching. Zero LLM calls, instant execution.

Logging

Structured Task Logs

Every task gets a JSONL log with step-by-step actions and a verified_success vs unverified_success distinction. Query via /task-logs.

Independence

Fully Standalone

No longer coupled to OpenClaw. Data lives in ~/.clawd-cursor/ (auto-migrates). Works with any AI, any client, any framework.

Tools

33 tools. Every desktop action covered.

GET /tools returns all schemas in OpenAI function-calling format. Any model that can call functions can use them.

Get Started

Up and running in 2 minutes.

Install once. Use from any AI — full agent, tools server, or MCP.

PowerShell
git clone https://github.com/AmrDab/clawd-cursor.git --branch v0.7.0
cd clawd-cursor
npm install; npm run setup
# Full agent
clawdcursor start
# Tools-only (bring your own AI)
clawdcursor serve
# MCP stdio
clawdcursor mcp
Terminal
git clone https://github.com/AmrDab/clawd-cursor.git --branch v0.7.0
cd clawd-cursor && npm install && npm run setup
# System Settings → Privacy → Accessibility → Add Terminal
chmod +x scripts/mac/*.sh scripts/mac/*.jxa
clawdcursor start  # or: serve | mcp
Terminal
git clone https://github.com/AmrDab/clawd-cursor.git --branch v0.7.0
cd clawd-cursor && npm install && npm run setup
# Linux: browser/CDP only (no native desktop automation yet)
clawdcursor start  # or: serve | mcp

Requires: Node.js 20+. npm run setup builds and registers clawdcursor globally.
