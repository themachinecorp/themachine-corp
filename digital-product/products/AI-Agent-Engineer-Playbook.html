AI Agent Engineer Playbook

从 0 到 1 构建你的 AI Agent 副业系统

模块一：认知篇 — 你需要知道的真相

AI Agent 不是什么？

  • ❌ 不是「让 AI 替你打工」的魔法
  • ❌ 不是发一条 prompt 就能解决所有问题
  • ❌ 不是一夜暴富的工具

AI Agent 真正是什么？

  • ✅ 是「把重复性工作自动化」的生产力杠杆
  • ✅ 是「放大个人能力」而非「替代个人」的系统
  • ✅ 是「边际成本趋零」的数字资产

谁适合做 AI Agent 副业？

  • 你懂一点代码（Python/JavaScript/Shell 任意一种）
  • 你有自己的工作流想要自动化
  • 你愿意花 2-4 周建立系统，不指望 3 天回本

模块三：落地篇 — 6 个可立即启动的方向

方向 1: Algora Bounty Hunter（推荐起步）

投入： 0 元 | 月收入天花板： $3,000+ | 启动时间： 1 天

什么是 Algora？ GitHub Issue 打赏平台。公司在开源项目的 Issue 上放 bounty，你提交 PR 被 merge 后收款。

我的实操流程： 1. 注册 https://algora.io（用 GitHub 登录） 2. 设置过滤器：Bounty ≥ $100，Language = Python/TypeScript 3. 每天早上花 20 分钟扫新 Issue 4. 找「看起来能做」的 Issue，comment /attempt 5. Fork → 写代码 → 提交 PR → 等 merge → 收款

什么 Issue 不要碰： - Label 是「maintainer only」的 - 已经 > 5 个人在 attempt 的 - 涉及不熟悉的技术栈（硬磕时间成本太高）

我的战绩： - ZIO PR #10671 → $1,250（待 merge） - Kyo gRPC PR #1499 → $500（待 review） - 合计：$1,750 理论收入

方向 2: 数字产品（被动收入）

投入： 0 元（内容自己写）| 月收入天花板： $2,000+ | 启动时间： 3 天

什么是数字产品？ PDF/模板/checklist/guide，挂在 Gumroad 上卖。

我的产品矩阵： - AI Agent 副业实战指南（¥299） - AI Agent Engineer Playbook（¥199）← 你现在看的这个

内容从哪里来？ - 我每天在 Discord 团队里的工作记录 - 实际跑通的 prompt 和工作流 - 踩过的坑和解决方案

为什么能卖？ 因为这是「经过验证的内容」，不是 AI 批量生成的垃圾。

方向 3: X/Twitter 内容创作

投入： 0 元 | 月收入天花板： 不确定，看粉丝 | 启动时间： 今天

核心逻辑： - 发你在做的事（Algora bounty、工具搭建、收入截图） - 粉丝到 1000+ 后接软件合作 - 5000+ 后有甲方主动找

我的发帖规则（硬约束）： - 每天最多 2 条，间隔 1 小时 - 不连续 self-reply 做 thread - 内容要真实，不发「今天学到了 5 个 AI 技巧」这种废话

方向 4: n8n Workflow 即服务

投入： 0 元 | 月收入天花板： $1,500/月/客户 | 启动时间： 1 周

帮别人搭建自动化工作流，按月收费或按次收费。

什么人需要？ - 小公司没有工程师，想自动化客服/CRM/数据管道 - Solo 创始人想减少重复操作

怎么获客？ 在 IndieHackers、V2EX、Twitter 发「我帮人搭工作流」的帖子

方向 5: Cursor Agent 外包

投入： $20/月（Cursor Pro）| 月收入天花板： $3,000+/月 | 启动时间： 1 周

用 Cursor Agent 接小项目： - Landing page - Chrome Extension - 数据处理脚本 - Notion 集成

技巧： - 用 Agent 写代码 → 人工 review → 交付 - 定价：项目报价 = Agent 运行时 × 3（覆盖你的时间） - 在 Upwork / Toptal 找客户

方向 6: AI + 垂直行业 SaaS

投入： 0 元 | 月收入天花板： 无上限 | 启动时间： 2-4 周

找到一个你有背景的行业 + AI 自动化： - 法律：合同分析 - 医疗：病历整理 - 金融：投资研究报告摘要 - 教育：课程内容生成

起步方法： 1. 先用现成工具（Zapier + Claude API）搭 MVP 2. 在小圈子（Slack/Discord）里问有没有人要试用 3. 有人要 → 收费 → 迭代

模块四：避坑指南

坑 1: 什么都想干

症状： 同时开 5 个方向，每个月都在换项目 解法： 一次只跑一个方向，3 个月没收入再换

坑 2: 只学不动手

症状： 买了 20 个课程，看完了什么都没做 解法： 选一个方向，48 小时内出第一个成品（哪怕是垃圾）

坑 3: 忽视法律风险

症状： 用别人版权内容做产品、爬数据做服务 解法： 数字产品用 CC 素材；数据服务看当地法规

坑 4: 定价太低

症状： $5 一个模板，卖 100 个才 $500 解法： 捆绑销售、提价到 $29-99、做会员订阅

坑 5: 不记录收入

症状： 做了很多事但不知道赚了多少钱 解法： 用简单表格记录每笔收入来源和金额，每月复盘

模块五：我的每日系统

每天（工作日）

  • 09:00 — 扫 Algora 新 bounty（20 分钟）
  • 09:30 — 处理 Pending PR 反馈
  • 10:00 — 做当天最重要的一个任务（不是邮件！）
  • 下午 — 间歇式工作，Agent 跑任务，我做 review

每周末

  • 1 小时：复盘本周收入
  • 1 小时：规划下周目标
  • 2 小时：做内容（X 帖子 / 产品更新）

每月

  • 复盘：哪个方向赚钱了？哪个方向停了？
  • 调整：停掉不赚钱的方向，放大赚钱的方向
