# FORGE+ 功能规划

## 产品定位
FORGE 的升级版，支持更长内容生成

## 新增功能

### 1. 长文生成 (Long-form)
- 博客文章 (2000-5000字)
- 深度分析报告
- 教程/指南

### 2. 电子书生成 (Ebook)
- 章节结构自动规划
- 完整大纲生成
- 自动填充内容

### 3. 批量生成 (Batch)
- 一次生成多篇
- 批量改写风格

## 定价
- FORGE PRO ($5): 现有功能
- FORGE+ ($8): 包含长文 + 电子书

## 技术实现
- 调用 MiniMax API 时增加 token 限制
- 添加新的 UI tab
- 复用现有架构

## 优先级
1. 长文生成（最重要）
2. 电子书（次要）
3. 批量（后续）
