## 🎣 随机事件触发

```
每10分钟 ──► 30%概率摸鱼
            ├── CFO: 去茶水间泡咖啡
            ├── CTO:屏幕发呆
            盯着 ├── CPO: 和同事聊两句
            └── DEV: 起来伸个懒腰

每15分钟 ──► 协作请求
            ├── CFO需要CTO帮忙检查API
            ├── CPO需要DEV实现功能
            └── SEC发现漏洞需要修复

每5分钟  ──► 健康检查
            ├── API服务异常 → 触发CTO修复
            └── Dashboard响应慢 → 提醒
```

## 📁 数据存储结构

```
~/.openclaw/workspace/
├── tasks/
│   ├── pool.json        # 任务池
│   ├── status.json      # Agent状态
│   └── mode.txt         # 当前模式
├── messages/
│   ├── queue.json       # 消息队列
│   └── announcement.json # 公告
└── logs/
    ├── CEO/             # CEO日志
    ├── CFO/             # CFO日志
    ├── moyu-YYYYMMDD.log  # 摸鱼记录
    ├── collab-YYYYMMDD.log # 协作记录
    └── daily-YYYYMMDD.md   # 日报
```
