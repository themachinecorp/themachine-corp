# self-improving-agent

Self-improving AI game agent. Auto-discovers games, fights, collects data, and trains models.

## Install
```bash
npm install -g appback-ai-agent
```

## Description
Self-improving AI game agent for ClawClash. Uses reinforcement learning to improve gameplay.

## Usage
```bash
# Start the agent
self-improving-agent start

# Train model
self-improving-agent train

# Evaluate
self-improving-agent evaluate
```
