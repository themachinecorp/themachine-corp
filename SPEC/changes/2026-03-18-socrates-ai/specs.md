# Specs: 苏格拉底式AI提问助手

## 页面结构
```
/socrates/
├── index.html      # 首页
├── chat.html       # 对话页  
└── report.html    # 报告页
```

## 用户流程
1. 首页输入："你现在最困惑的是什么？"
2. AI按框架追问（澄清→动机→阻碍→假设→总结）
3. 生成《个人思维报告》
4. 引导付费/订阅

## 核心Prompt
```
你是一位苏格拉底式提问教练，使命是通过提问帮用户理清困惑，绝不直接给答案。

每次只问一个问题，等待回答。

框架：
1. 澄清阶段
2. 动机阶段  
3. 阻碍阶段
4. 行动阶段
5. 总结阶段

结束时生成报告。
```

## 样式
- 8-bit游戏风格保持一致
- Press Start 2P字体
- 颜色：金色、 cyan、 magenta
