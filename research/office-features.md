# Office 新功能规划

> 为 THEMACHINE Corp. Office 页面规划新功能
> 参考 Alex Finn 的 AI Agent 工作台风格

---

## 概述

Office 页面是 THEMACHINE 的核心指挥中心，展示 AI Agent 团队的工作状态、实时数据和任务调度。

**设计风格**: 复古终端 + 赛博朋克，结合 THEMACHINE 现有视觉语言

---

## 1. Agent 可视化展示

### 1.1 Agent 团队矩阵

```
┌─────────────────────────────────────────────────────────┐
│  🧠 AGENT STATUS MATRIX                                 │
├─────────┬─────────┬─────────┬─────────┬─────────┬──────┤
│  CEO    │  CTO    │  CFO    │  CMO    │  CPO    │ SEC  │
│  🟢     │  🟢     │  🟡     │  🔴     │  🟢    │ 🟢   │
│  Active │  Active │  Idle   │  Error  │ Active  │Active│
└─────────┴─────────┴─────────┴─────────┴─────────┴──────┘
```

- **状态指示器**: 🟢在线/🟡忙碌/🔴离线/⚠️错误
- **悬停效果**: 显示 Agent 实时负载、内存使用
- **点击交互**: 展开 Agent 详情面板

### 1.2 工作流可视化

- **任务管道**: 显示任务在各 Agent 间的流转
- **连接线动画**: 任务传递时的流光效果
- **实时状态**: 任务完成度、耗时、队列长度

### 1.3 Agent 3D 头像 (可选)

- 使用 Three.js 或纯 CSS 实现
- 每个 Agent 有独特的像素风格头像
- 空闲时微动画，工作时高亮

## 2. 实时数据展示

### 2.1 核心指标面板

| 指标 | 数值 | 趋势 |
|------|------|------|
| 活跃 Agent | 5/6 | ↑ |
| 今日任务 | 247 | ↑12% |
| 响应时间 | 1.2s | ↓ |
| API 调用 | 1,842 | ↑ |

### 2.2 实时图表

- **任务吞吐量**: 折线图，每分钟更新
- **资源使用率**: 环形图 (CPU/Memory/带宽)
- **Agent 负载分布**: 柱状图

### 2.3 滚动日志终端

```
[22:50:01] CEO → 分配任务给 CTO
[22:50:02] CTO → 开始处理请求 #1847
[22:50:03] CFO → 验证交易数据
[22:50:04] CMO → 生成内容策略
[22:50:05] CTO → 任务完成，耗时 2.3s
```

- 彩色高亮不同级别日志
- 自动滚动，最新在最前
- 可过滤、搜索

## 3. 用户交互功能

### 3.1 任务发布面板

- **快速发布**: 输入框 + Agent 选择下拉
- **任务模板**: 预设常用任务类型
- **优先级设置**: 🔴紧急/🟡高/🟢普通

### 3.2 Agent 控制

- **启停开关**: 一键启停单个 Agent
- **负载调节**: 调整 Agent 处理速度
- **手动触发**: 强制执行特定任务

### 3.3 仪表盘定制

- 拖拽组件布局
- 显示/隐藏特定模块
- 主题切换 (暗色/亮色/复古)

## 6. 技术实现建议

### 6.1 前端

- **框架**: Next.js (现有)
- **图表**: Recharts / Chart.js
- **状态管理**: React Query (实时数据)
- **WebSocket**: 实时日志推送

### 6.2 后端

- **API**: 现有 `/api/agents` 扩展
- **数据库**: SQLite (现有) + 时序数据
- **缓存**: Redis (实时状态)

### 6.3 数据流

```
User Action → WebSocket → Server → Agent → Result → WebSocket → UI Update
                                    ↓
                              [Log to DB]
```
